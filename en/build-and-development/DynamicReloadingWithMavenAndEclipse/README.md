Dynamic Reloading With Maven And Eclipse
========================================

Abstract
--------

This example shows how a single-module Maven project has to be configured to enable an automatic reloading of the web application by a running Jetty instance when changes are made to the project's sources.

- - - - - - - - -

* __Tutorial__: [Dynamic Reloading of Vaadin Applications With Maven and Eclipse, Pt. I](http://blog.oio.de/2012/08/23/dynamic-reloading-of-vaadin-applications-with-maven-and-eclipse/)

* __Author__: Roland Krüger

* __Publishing Date__: August 23rd, 2012

* __Tutorial Language__: English

* __Vaadin Version__: any

Build
-----

To run the example, [Apache Maven](http://maven.apache.org) is needed. Run the example as described in the tutorial.
